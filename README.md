# UNIFIED MULTIVERSE

The prior question to Unified Mechanics: where does the axiom come from?

UM takes c² = c + 1 as a first principle. This repo asks what two structures, when their boundaries touch, produce that axiom as the unique stable interface recursion.

## The answer

Two parents:

| Parent | Transformation | Fixed point | Character |
|---|---|---|---|
| A (Accumulator) | x → x + 1 | none | Pre-time matter. No light. No equilibrium. |
| B (Inverter) | x → 1/x | x* = 1, r = 0.5 | Self-referential. Boundary-dominant. Symmetric. |

Their interface: x = 1 + 1/x → **x² = x + 1**

Our universe is the child. The one finite stable recursion that satisfies both parents simultaneously.

## Parent B empirical values

- r_B = 0.5 (vs our 0.309)
- W_L = W_M = 0.25 (symmetric, vs our 0.4775 / 0.0955)
- W_B = 0.50 (boundary-dominant)
- ε_floor = 12.5% (four times ours)

## What dark matter is

Both parents are present. They cannot be detected electromagnetically — Parent A has no light channel, Parent B's light runs under a different axiom. Their gravitational influence presses in through geometry. This is dark matter: the two structures that made us, still flanking us, visible only through curvature.
